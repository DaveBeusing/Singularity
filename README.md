# Singularity

Singularity is a lightweight system stress testing and hardware validation platform written entirely in C#.

The application can generate configurable load on:

* CPU
* Memory
* GPU

while simultaneously monitoring system metrics in real time.

## Features

* Real-time telemetry
* CPU stress testing
* Memory stress testing
* GPU stress testing
* Portable deployment
* No external runtime dependencies
* Clean MVVM architecture
* Logging and reporting

## Project Goals

The primary goal of Singularity is educational:

Every subsystem is implemented from scratch to better understand how hardware stress testing and telemetry systems work internally.
